# **SOLIDOS DE REVOLUCION**
Se usó python para la mostrar un concepto de calculo integral, **solidos de revolución**.  
Por definición, si una región R en un plano *xy*, se hace girar en torno a un eje *L*, se generá el solido de revolucion.  

En el codigo definimos una funcion $f(x) = x^2 + 1$ (una parábola desplazada hacia arriba).
